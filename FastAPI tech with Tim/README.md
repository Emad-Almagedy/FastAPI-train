1. using uv create the intial files `uv init .`
2. copy the .env variables from `imagekit.io` after creating the account 
3. to run the app from the terminal `uv run .\main.py`
4. you can use /docs or /redoc to check the FastAPI endpoints and test them 
5. to run the frontend made with steamlit use uv run streamlit run frontend.py
